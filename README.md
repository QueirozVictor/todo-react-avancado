# Todo List React

Aplicação de lista de tarefas desenvolvida com React como parte dos estudos de Hooks, Context API, Hooks Customizados e Memoization.

## Funcionalidades

* Adicionar novas tarefas
* Marcar tarefas como concluídas
* Remover tarefas
* Filtrar tarefas por:

* Todas
* Pendentes
* Concluídas
* Persistência de dados utilizando Local Storage

## Tecnologias Utilizadas

* React
* Vite
* JavaScript
* Context API
* React Hooks
* Tailwind CSS
* Local Storage

## Conceitos Aplicados

* useState
* useEffect
* useContext
* useMemo
* React.memo
* Hooks Customizados

## Como Executar o Projeto Localmente

1. Clone o repositório:

```bash
git clone https://github.com/QueirozVictor/todo-react-avancado.git
```

2. Acesse a pasta do projeto:

```bash
cd to-do-list-app
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm run dev
```

5. Abra o navegador no endereço:

```txt
http://localhost:5173
```
